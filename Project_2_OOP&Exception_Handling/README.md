# OOP Basics & Error Handling

## About This Project
Building intelligent RPA systems requires a solid architecture and fail-safe mechanisms. This project provides simple and clear examples of Object-Oriented Programming (OOP) combined with effective error management techniques.

## Key Concepts Covered
* Creating Classes and Objects.
* Attributes and Methods.
* Try-Catch blocks (Exception Handling) to prevent unexpected system crashes during automation.

*You will find the notebook divided internally into logical operations that you can run sequentially.*
