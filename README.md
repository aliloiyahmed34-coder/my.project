# Robotic Process Automation (RPA)

This repository contains three independent projects, each serving the same fundamental idea: **designing and programming intelligent systems to automate routine electronic tasks**, which significantly reduces human intervention and minimizes manual errors.

## Available Projects:

### 1. [Core Functions](./Project_1_Functions)
This section contains essential functions that act as the main tools of programming and the cornerstone upon which intelligent systems are built.

### 2. [OOP Basics & Error Handling](./Project_2_OOP&Exception_Handling)
Here you will find practical examples of Object-Oriented Programming (OOP) and error management (Exception Handling), which are crucial aspects of building robust and reliable systems.

### 3. [Applied OOP Project](./Project_3_OOP2)
This folder contains a simple, integrated project built entirely on Object-Oriented Programming principles, serving as a solid foundation for developing more complex systems in the future.
